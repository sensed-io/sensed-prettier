# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-08-01

### Added

- Initial Prettier configuration
- Support for JavaScript, TypeScript, JSX, and common web formats
- Comprehensive `.prettierignore` with common patterns
- Documentation and usage examples
- MIT License

### Configuration Details

- Print width: 80 characters
- Tab width: 2 spaces
- Use spaces instead of tabs
- Always include semicolons
- Use single quotes
- ES5-compatible trailing commas
- Add bracket spacing
- Avoid arrow function parentheses when possible
- Use LF line endings
